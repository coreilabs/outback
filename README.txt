CARDÁPIO MOBILE BILÍNGUE — HTML, CSS E JAVASCRIPT

Arquivos:
- index.html: estrutura da página
- styles.css: visual mobile-first e responsivo
- script.js: dados dos pratos, tradução, filtros, busca, favoritos e modal

Como abrir:
1. Extraia o arquivo ZIP.
2. Abra o arquivo index.html no navegador.
3. Não precisa instalar nada.

Como editar os pratos:
- Abra script.js.
- Localize a constante menuItems.
- Altere nome, tradução, descrição, categoria e cores de cada item.

Como inserir preços:
- Adicione uma propriedade price em cada objeto de menuItems.
- No template do index.html, crie um elemento para exibir o preço.
- No renderMenu() do script.js, preencha esse elemento com item.price.

Observação:
Projeto demonstrativo e não oficial. Os nomes e categorias foram usados como referência de navegação. Disponibilidade, composição e preços podem variar conforme a unidade.
